# Training Scenario 01: Emergency Wallet Freeze

## Scenario Title

Execution of Emergency Wallet Freeze for Sanctions Compliance.

## Learning Objectives

- Correctly identify the target wallet address.
- Execute the freeze command using the Sovereign Control Portal or API.
- Verify the frozen state on the blockchain.
- Understand the audit trail generated by the action.

## Preconditions

- Operator has `ECB_ADMIN` role.
- Operator possesses a valid `ISSUING` key.
- Access to the Closed Settlement Plane (CSP) is established.

## Initial System State

- Target wallet `0x71C7656EC7ab88b098defB751B7401B5f6d8976F` is active.
- Balance is â‚¬1,250.00.
- System status is `ACTIVE`.

## Triggering Event

Receipt of an official legal order (Case ID: 2026-001) requiring the immediate freezing of the target wallet due to sanctions listing.

## Step-by-Step Actions

1. **Authentication**: Log in to the Sovereign Control Portal using mTLS and the `ISSUING` key.
2. **Identification**: Enter the target address `0x71C7656EC7ab88b098defB751B7401B5f6d8976F` in the Sanctions UI.
3. **Justification**: Enter the mandatory justification: "Emergency Sanctions Compliance - Case ID: 2026-001".
4. **Execution**: Click "Freeze Account" and confirm the action in the safety modal.
5. **Verification**: Query the account status to confirm `frozen: true`.

## Expected System Responses

- API returns `200 OK` with a transaction hash.
- Blockchain emits `AccountFrozen` event.
- Audit log records `ACCOUNT_FROZEN` event.

## Common Mistakes to Avoid

- Entering the wrong wallet address (always use copy-paste).
- Forgetting to include the legal reference in the justification.
- Failing to verify the state change after execution.

## Audit Artifacts Produced

- `ACCOUNT_FROZEN` log entry in the Audit Service.
- Blockchain transaction receipt.
- Signed entry in the Governance Service linking the action to the `ISSUING` key.

## Completion Criteria

The target wallet is successfully frozen, and the action is verified through both the UI and a direct blockchain query.
