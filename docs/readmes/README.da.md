# tEUR — Tokenized Euro (tEUR) Repository

Repository med smart contracts, API-gateway, dashboard og værktøjer til Tokenized Euro-projektet.

Hurtig start

- Kør API-tests:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Start reconciliation-reference:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Se hoved-README og `docs/` for detaljerede instruktioner.
