# tEUR — Tokenized Euro (tEUR) Repository

Tároló, amely a Tokenized Euro projekthez tartalmaz smart contractokat, API-gateway-t, irányítópultot és eszközöket.

Gyors indítás

- API tesztek futtatása:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Reconciliation referencia futtatása:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Részletes utasításokért lásd a fő README-t és a `docs/` könyvtárat.
