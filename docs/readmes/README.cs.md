# tEUR — Tokenized Euro (tEUR) Repository

Repozitář obsahující smart kontrakty, API bránu, dashboard a nástroje pro projekt Tokenized Euro.

Rychlé spuštění

- Spustit testy API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Spustit referenční reconciliaci:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Podrobné pokyny naleznete v hlavním README a v `docs/`.
