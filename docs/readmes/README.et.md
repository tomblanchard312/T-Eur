# tEUR — Tokenized Euro (tEUR) Repository

Hoop, mis sisaldab smart contract'e, API-väravat, armatuurlaua ja tööriistu Tokenized Euro projekti jaoks.

Kiirstart

- Käivita API testid:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Käivita reconciliatsiooni referents:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Vaata üksikasjade jaoks peamist README-d ja `docs/`.
