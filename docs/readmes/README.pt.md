# tEUR — Tokenized Euro (tEUR) Repository

Repositório contendo smart contracts, gateway API, painel e ferramentas para o projeto Tokenized Euro.

Início rápido

- Executar testes da API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Executar referência de reconciliação:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Consulte o README principal e `docs/` para instruções detalhadas.
