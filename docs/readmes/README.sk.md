# tEUR — Tokenized Euro (tEUR) Repository

Repozitár obsahujúci smart kontrakty, API gateway, dashboard a nástroje pre projekt Tokenized Euro.

Rýchly štart

- Spustiť API testy:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Spustiť referenciu reconciliácie:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Pozri hlavný README a `docs/` pre podrobné inštrukcie.
