# tEUR — Tokenized Euro (tEUR) Repository

Shramba, ki vsebuje pametne pogodbe, API prehod, nadzorno ploščo in orodja za projekt Tokenized Euro.

Hiter začetek

- Zaženi API teste:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Zaženi referenčno storitev za uskladitev:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Oglej si glavni README in `docs/` za podrobna navodila.
