# tEUR — Tokenized Euro (tEUR) Repository

Varasto, joka sisältää smart contractit, API-portin, kojetaulun ja työkalut Tokenized Euro -projektille.

Pika-aloitus

- Suorita API-testit:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Käynnistä reconciliaton referenssi:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Katso yksityiskohtaiset ohjeet pää-README:stä ja `docs/`-kansiosta.
