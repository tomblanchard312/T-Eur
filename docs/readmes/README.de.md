# tEUR — Tokenized Euro (tEUR) Repository

Repository mit Smart Contracts, API-Gateway, Dashboard und Tools für das Tokenized Euro Projekt.

Schnellstart

- API-Tests ausführen:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Reconciliation-Referenz starten:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Siehe das Haupt-README und `docs/` für ausführliche Anweisungen.
