# tEUR — Tokenized Euro (tEUR) Repository

Repository som innehåller smarta kontrakt, API-gateway, dashboard och verktyg för Tokenized Euro-projektet.

Snabbstart

- Kör API-tester:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Kör rekonsilieringsreferens:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Se huvud-README och `docs/` för detaljerade instruktioner.
