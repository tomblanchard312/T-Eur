# tEUR — Tokenized Euro (tEUR) Repository

Repositorio que contiene contratos inteligentes, API gateway, panel y herramientas para el proyecto Tokenized Euro.

Inicio rápido

- Ejecutar pruebas de la API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Ejecutar la referencia de conciliación:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Consulte el README del repositorio y `docs/` para más detalles.
