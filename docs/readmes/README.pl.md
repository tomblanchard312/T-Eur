# tEUR — Tokenized Euro (tEUR) Repository

Repozytorium zawierające smart kontrakty, bramę API, pulpit i narzędzia projektu Tokenized Euro.

Szybki start

- Uruchom testy API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Uruchom serwis rekonsyliacyjny referencyjny:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Zobacz główny README i `docs/` po szczegółowe instrukcje.
