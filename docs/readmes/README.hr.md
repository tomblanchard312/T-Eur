# tEUR — Tokenized Euro (tEUR) Repozitorij

Repozitorij koji sadrži pametne ugovore, API gateway, nadzornu ploču i alate za projekt Tokenized Euro.

Brzi početak

- Pokreni API testove:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Pokreni referentnu reconciliaciju:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Pogledajte glavni README i `docs/` za detaljne upute.
