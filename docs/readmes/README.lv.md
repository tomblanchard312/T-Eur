# tEUR — Tokenized Euro (tEUR) Repository

Krātuve, kas satur smart līgumus, API vārteju, paneli un rīkus Tokenized Euro projektam.

Ātra sākšana

- Palaist API testus:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Palaist conciliācijas referenci:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Skatīt galveno README un `docs/` sīkākiem norādījumiem.
