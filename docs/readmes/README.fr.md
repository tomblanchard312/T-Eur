# tEUR — Tokenized Euro (tEUR) Repository

Dépôt contenant les smart contracts, la passerelle API, le tableau de bord et les outils du projet Tokenized Euro.

Démarrage rapide

- Exécuter les tests API :
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Lancer le service de réconciliation de référence :
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Consultez le README principal et `docs/` pour les instructions détaillées.
