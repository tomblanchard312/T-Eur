# tEUR — Tokenized Euro (tEUR) Repository

Saugykla, kurioje yra smart kontraktai, API tarpininkas, prietaisų skydelis ir įrankiai Tokenized Euro projektui.

Greitas pradėjimas

- Paleisti API testus:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Paleisti reconciliacijos referencinį servisą:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Daugiau informacijos rasite pagrindiniame README ir `docs/`.
