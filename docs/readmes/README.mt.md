# tEUR — Tokenized Euro (tEUR) Repository

Repository li għandu smart contracts, API gateway, dashboard u għodda għal proġett Tokenized Euro.

Bidu malajr

- Esegwixxi t-testijiet tal-API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Ibda r-referenza tar-rekonsiljazzjoni:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Ara r-README prinċipali u `docs/` għall-istruzzjonijiet dettaljati.
