# tEUR — Tokenized Euro (tEUR)

Tá stór ina bhfuil conarthaí cliste, geata API, painéal agus uirlisí don tionscadal Tokenized Euro.

Tús tapa

- Rith tástálacha API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Rith seirbhís comhtháthaithe tagartha:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Féach an README príomhúil agus `docs/` le haghaidh treoracha mionsonraithe.
