# tEUR — Tokenized Euro (tEUR) Repository

Repository care conține contracte smart, gateway API, panou de control și unelte pentru proiectul Tokenized Euro.

Pornire rapidă

- Rulează testele API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Rulează serviciul de reconciliere de referință:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Consultă README principal și `docs/` pentru instrucțiuni detaliate.
