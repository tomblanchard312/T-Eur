# tEUR — Tokenized Euro (tEUR) Repository

Αποθετήριο που περιέχει smart contracts, gateway API, πίνακα ελέγχου και εργαλεία για το έργο Tokenized Euro.

Γρήγορη εκκίνηση

- Εκτέλεση δοκιμών API:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Εκτέλεση αναφοράς συμφιλίωσης:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Δείτε το κύριο README και το `docs/` για λεπτομερείς οδηγίες.
