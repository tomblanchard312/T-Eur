# tEUR — Tokenized Euro (tEUR) Репозитория

Репозитория съдържа смарт договори, API gateway, табло и инструменти за проекта Tokenized Euro.

Бърз старт

- Стартиране на API тестовете:
  ```bash
  cd api
  npm ci
  npx vitest
  ```
- Стартиране на референтния reconciliation service:
  ```bash
  cd tools/reconciliation-ref
  node index.js
  ```

Вижте основния README и `docs/` за подробни инструкции.
